# NoSi
A simple extension that removes the ?si= tracking parameter from YouTube links.

---
## HOW IT WORKS:
Download the extension and import it into your browser. Then, view a YouTube video and copy the URL from the "Share" tab. It'll remove the **?=si** tracking parameter! **Works if you use the Copy button, right-click and copy, or CTRL+C**.
<img width="1920" height="1080" alt="demonstration" src="https://github.com/user-attachments/assets/3ce48c8e-b83d-454c-9a4a-a367a689f79d" />

Clipboard data is **NOT** sent or shared **ANYWHERE**, and is **ENTIRELY** local.
